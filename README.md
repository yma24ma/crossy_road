# Crossy Road
## What this project is about
This is a Crossy Road–style game built with **HTML5 Canvas + vanilla JavaScript**. The goal is to keep hopping forward as long as possible while avoiding hazards like cars and trains, and using logs to cross rivers.

## How to play
**Goal:** Survive and get the highest score by moving forward.

### Desktop controls
- **Arrow keys / WASD**: move
- **Enter**: start / play again
- **R**: restart

### Mobile controls
- **Swipe** to move (direction is based on the chicken’s perspective)
- **Tap**: start + hop forward
